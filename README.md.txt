# 🛡️ Defense Sentinel – AI Intrusion Detection

Real-time surveillance app using YOLOv8 and Streamlit to detect and log human intrusions.

---

## 🔍 Features
- 🧠 Real-time person detection using YOLOv8
- 🎥 Live webcam feed with detection overlays
- 📝 Intrusion logs with timestamped screenshots
- 📁 Saves intrusions to `/intrusions/` folder
- 🧑‍💻 Built with Streamlit, OpenCV, PIL, and Ultralytics

---
